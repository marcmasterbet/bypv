1. Données collectées

ByePV collecte les données suivantes :

  • Nom, prénom, adresse, email, téléphone (pour générer la lettre)
  • Informations relatives à l'avis de contravention
  • Photo du PV (traitée par l'API OpenAI, non stockée)

2. Utilisation des données

Les données sont utilisées exclusivement pour :

  • Générer votre lettre de contestation
  • Traiter votre paiement via Stripe

3. Stockage

Aucune donnée personnelle n'est stockée sur nos serveurs après la génération de la lettre.

4. Partage des données

Vos données peuvent être partagées avec :

  • OpenAI (analyse de l'image du PV)
  • Stripe (traitement du paiement)

Ces prestataires disposent de leurs propres politiques de confidentialité.

5. Vos droits

  • Droit d'accès
  • Droit de rectification
  • Droit à l'effacement

Pour exercer vos droits : support.byepv@gmail.com

6. Cookies

L'application n'utilise pas de cookies.
